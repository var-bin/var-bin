Как начать использовать viewport

  • Какие еще свойства можно задавать для viewport
    1. minimum-scale - устанавливает минимальное значение zoom
    2. maximum-scale - устанавливает максимальное значение zoom
    3. height - можно привязываться к высоте, но смысла от этого немного больше, чем 0
    4. user-scalable - определяет, доступна ли возможность zoom. Если равно no, то пользователь не сможет исрользовать возможность zoom. Используйте только тогда, когда это реально нужно

Медиазапросы @media

Применяем стили на основе размера viewport
Параметр Описание
min-width Правила применяются для любого браузера/устройства, когда width больше, чем определенно в запросе
max-width Правила применяются для любого браузера/устройства, когда width меньше, чем определенно в запросе
min-height Правила применяются для любого браузера/устройства, когда height больше, чем определенно в запросе
max-height Правила применяются для любого браузера/устройства, когда height меньше, чем определенно в запросе

Медиазапросы @media

Применяем стили на основе размера viewport
Параметр Описание
orientation=portrait Правила применяются для любого браузера/устройства, когда height больше или равен width
orientation=landscape Правила применяются для любого браузера/устройства, когда width больше чем height
