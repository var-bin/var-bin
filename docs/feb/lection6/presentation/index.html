    1. Все элементы с классом .grid__item должны быть по ширине 300px, по высоте не должно быть никаких ограничений.
    2. Все элементы с классом .grid__item должны иметь внутренний отступ со всех сторон по 10px. Также внешний отступ по 10px со всех сторон. Обратите внимание на Collapsed margin.
    3. Все элементы с классом .grid__item должны иметь рамку вокруг размером 2px, цвет и тип на ваш выбор. Также задайте радиус для данной рамки 3px.
