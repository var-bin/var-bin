Как использовать CSS

  • Внешний файл <link rel="stylesheet" href="style.css">
  • В теге <head> при помощи тега <style>
  • Инлайн стили при помощи тега <style>

Как использовать CSS. Внешний файл.

      <head>
        <title>External stylesheet</title>
        <link rel="stylesheet" href="styles.css"/>
      </head>
      /* styles.css */
      h1 {
        color: blue;
        background-color: yellow;
      }

Как использовать CSS. В теге <head>

      <head>
        <title>Internal stylesheet</title>
        <style>
          h1 {
            color: blue;
            background-color: yellow;
          }
      </style>
      </head>

Как использовать CSS. Инлайн стили при помощи тега <style>

      <body>
        <h1 style="color: blue; background-color: yellow;">
          Lorem ipsum dolor sit amet.
        </h1>
      </body>

Наследование

      h1 {
        color: inherit;
      }

Подключение других стилей внутрь styles.css

      @import "mystyle.css";
      @import url("mystyle.css");
      @import url("fineprint.css") print;
      @import url("bluish.css") projection, tv;

Шрифты. @font-face

      /* regular face of Gentium */
      @font-face {
        font-family: MyGentium;
        /* Используем локальный шрифт Gentium */
        src: local(Gentium),
        /* Иначе, загружаем его по сети */
             url(Gentium.woff);
      }
