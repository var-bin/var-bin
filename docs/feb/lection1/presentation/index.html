Обратите внимание

Optional tags

Для большого списка HTML элементов есть возможность не указывать открывающийся тег или закрывающийся тег. Важный момент, это не одинарные HTML элементы. Просто стандарт предусматривает такую возможность. Я категорически вам не рекомендую пропускать открывающийся тег или закрывающийся тег, если такие должны быть у HTML элемента.

И последнее, но не менее важное

Синтаксические ошибки 
      <!-- example -->
      <p><i>He dreamt.
      <p><i>He dreamt that he ate breakfast.
      <p><i>Then lunch.
      <p><i>And finally dinner.
