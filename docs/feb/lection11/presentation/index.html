Что такое svg

Почему SVG?

  • Маленький размер файла, который хорошо сжимается
  • Масштабируется на любые размеры без потери качества
  • Отличный выбор для retina-экранов
  • Можно редактировать в любом текстовом редакторе
  • Анимация и фильтры

Базовые фигуры. Circle

      <svg>
        <circle r="50" cx="50%" cy="50%" fill="orange"/>
      </svg>
  • r - Задает радиус круга. Если не указывать единицы измерения, то по умолчанию будут px. Не может принимать отрицательные значения. Если значение этого атрибута равно 0 - круг не будет отображен.
  • cx - Задает координаты оси X относительно центра круга. Если не указать этот атрибут, то его значение будет равно 0.
  • cy - Задает координаты оси Y относительно центра круга. Если не указать этот атрибут, то его значение будет равно 0.

Базовые фигуры. Rect

      <svg>
        <rect width="50%" height="100" fill="yellowgreen" x="25%" 
        y="25" stroke="grey" stroke-width="10"/>
      </svg>
  • width - Задает ширину прямоугольника. Не может принимать отрицательные значения. Если значение этого атрибута равно 0 - прямоугольник не будет отображен.
  • height - Задает высоту прямоугольника. Не может принимать отрицательные значения. Если значение этого атрибута равно 0 - прямоугольник не будет отображен.

Базовые фигуры. Rect

  • x - Задает координаты оси X. Если не указать этот атрибут, то его значение будет равно 0.
  • y - Задает координаты оси Y. Если не указать этот атрибут, то его значение будет равно 0.
  • rx - Задает скругление углов по оси X. Если не указать ry, то значение этого атрибута будет равно rx.
  • ry - Задает скругление углов по оси Y. Если не указать rx, то значение этого атрибута будет равно ry.

Базовые фигуры. Polygon

      <svg>
        <polygon fill="red" stroke="yellowgreen" stroke-width="10" 
          points="350,75 379,161 469,161 397,215 423,301 350,250
                 277,301 303,215 231,161 321,161"/>
      </svg>
  • points - набор точек, которые задают многоугольник.

Базовые фигуры. Polyline

        <polyline fill="none" stroke="purple" stroke-width="10"

  • points - набор точек, которые задают ломаную линию.

Базовые фигуры. Line

      <svg>
        <line x1="10" y1="300" x2="100" y2="10" stroke-width="5"/> 
      </svg>
  • x1 - Задает координаты оси X относительно начала линии. Если не указать этот атрибут, то его значение будет равно 0.
  • y1 - Задает координаты оси Y относительно начала линии. Если не указать этот атрибут, то его значение будет равно 0.
  • x2 - Задает координаты оси X относительно конца линии. Если не указать этот атрибут, то его значение будет равно 0.
  • y2 - Задает координаты оси Y относительно конца линии. Если не указать этот атрибут, то его значение будет равно 0.

Базовые фигуры. Ellipse

      <svg>
        <ellipse cx="50%" rx="70" ry="30" fill="red"/> 
      </svg>
  • cx - Задает координаты оси X относительно центра элипса. Если не указать этот атрибут, то его значение будет равно 0.
  • cy - Задает координаты оси Y относительно центра элипса. Если не указать этот атрибут, то его значение будет равно 0.

Базовые фигуры. Ellipse

  • rx - Задает радиус относительно оси X. Если не указать этот атрибут, то его значение будет равно 0. Не может принимать отрицательные значения.
  • ry - Задает радиус относительно оси Y. Если не указать этот атрибут, то его значение будет равно 0. Не может принимать отрицательные значения.

SVG Path - <path/>

  • d - содержит параметры пути (координаты).

Обозначения:

  • M - "moveto" команда, которая определяет новую точку.
  • L - "lineto" команда, которая "рисует" прямую линию с текущей точки в новую точку.
  • H - горизонтальная "lineto" команда, которая "рисует" прямую линию с текущей точки в новую точку.

SVG Path - <path/>

  • V - вертикальная "lineto" команда, которая "рисует" прямую линию с текущей точки в новую точку.
  • ...
  • Z - "closepath" завершает текущий подпуть и вызывает автоматическую прямую линию, которая будет нарисована от текущей точки до начальной точки текущего подпути.

Практика

SVG. Пишем текст. text

      <svg>
        <text x="15%" y="60%" font-family="Arial" font-size="55"
        fill="red">
          Hello, out there!
        </text>
      </svg>
  • x - Если одно значение, то задает координаты по оси X для первого глифа. Если значения указаны через запятую или через пробел, то каждое значение задает позицию для каждого отдельно взятого глифа по оси X.

SVG. Пишем текст. text

  • y - Если одно значение, то задает координаты по оси Y для первого глифа. Если значения указаны через запятую или через пробел, то каждое значение задает позицию для каждого отдельно взятого глифа по оси Y.
  • rotate - позволяет "вращать" текст. Если значения указаны через запятую или через пробел, то каждое значение задает "вращение" для каждого отдельно взятого глифа.

SVG. Пишем текст. Практика

Напишите текст "Front-End Basic". Слово "Front" должно быть зеленого цвета. Слово "End" должно быть красного цвета. Слово "Basic" должно быть синего цвета. Коротное тире должно быть без заливки с обводкой в 2px, цвет обводки grey.
