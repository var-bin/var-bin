Что такое svg

Почему SVG?

  • Маленький размер файла, который хорошо сжимается
  • Масштабируется на любые размеры без потери качества
  • Отличный выбор для retina-экранов
  • Можно редактировать в любом текстовом редакторе
  • Анимация и фильтры

Базовые фигуры. Circle

      <svg>
        <circle r="50" cx="50%" cy="50%" fill="orange"/>
      </svg>
  • r - Задает радиус круга. Если не указывать единицы измерения, то по умолчанию будут px. Не может принимать отрицательные значения. Если значение этого атрибута равно 0 - круг не будет отображен.
  • cx - Задает координаты оси X относительно центра круга. Если не указать этот атрибут, то его значение будет равно 0.
  • cy - Задает координаты оси Y относительно центра круга. Если не указать этот атрибут, то его значение будет равно 0.

Базовые фигуры. Rect

      <svg>
        <rect width="50%" height="100" fill="yellowgreen" x="25%" 
        y="25" stroke="grey" stroke-width="10"/>
      </svg>
  • width - Задает ширину прямоугольника. Не может принимать отрицательные значения. Если значение этого атрибута равно 0 - прямоугольник не будет отображен.
  • height - Задает высоту прямоугольника. Не может принимать отрицательные значения. Если значение этого атрибута равно 0 - прямоугольник не будет отображен.

Базовые фигуры. Rect

  • x - Задает координаты оси X. Если не указать этот атрибут, то его значение будет равно 0.
  • y - Задает координаты оси Y. Если не указать этот атрибут, то его значение будет равно 0.
  • rx - Задает скругление углов по оси X. Если не указать ry, то значение этого атрибута будет равно rx.
  • ry - Задает скругление углов по оси Y. Если не указать rx, то значение этого атрибута будет равно ry.

Базовые фигуры. Polygon

      <svg>
        <polygon fill="red" stroke="yellowgreen" stroke-width="10" 
          points="350,75 379,161 469,161 397,215 423,301 350,250
                 277,301 303,215 231,161 321,161"/>
      </svg>
  • points - набор точек, которые задают многоугольник.

Базовые фигуры. Polyline

  • points - набор точек, которые задают ломаную линию.

Transform

Виды трансформации:

  • 2D
    • rotate()
    • scale()
    • translate()
    • skew()
  • 3D
    • rotateX()
    • rotateY()
    • scaleX()
    • scaleY()
    • translateX()
    • translateY()
  • skewX()
  • skewY()

Transform

rotate - вращение

      button {
        transform: rotate(360deg);
      }
      /* === */
      button {
        transform: rotate(1turn);
      }

Transform

scale - растягивает элемент

      button {
      /* Увеличиваем в 2 раза */
        transform: scale(2);
      }
      /* По горизонтали в 4, по вертикали в 2 раза раза */
      button {
        transform: scale(4, 2);
      }

Transform

translate - двигаем элемент

      button {
      /* Сдвинуть вправо на 3px */
        transform: translate(3px);
      }
      button {
      /* Сдвинуть вправо и вниз на 3px */
        transform: translate(3px, 3px);
      }

Transform

skew - искажаем элемент

      button {
      /* Второй параметр = 0 */
        transform: skew(10deg);
      }
      /* ... */
      button {
        transform: skew(10deg, 15deg);
      }

Animation. @keyframes

Определяем анимацию

      @keyframes example {
        from { transform: rotate(0); }
        to { transform: rotate(45deg); }
      }

Animation. @keyframes

Используем анимацию

      #someElement {
                /* name  duration       timing func */
        animation: example 2s 0s infinite ease;
                          /* delay  iteration */
      }

Animation. @keyframes

Делаем больше шагов

      @keyframes example2 {
        0% { transform: rotate(0); }
        10% { transform: rotate(10deg); }
        20% { transform: rotate(10deg); }
        40% { transform: rotate(40deg); }
        80% { transform: rotate(80deg); }
        100% { transform: rotate(120deg); }
      }
