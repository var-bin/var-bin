Спрайты. Что это и для чего

CSS-спрайт – способ объединить много изображений в одно, чтобы:

  • Сократить количество обращений к серверу.
  • Загрузить несколько изображений сразу, включая те, которые понадобятся в будущем.
  • Если у изображений сходная палитра, то объединённое изображение будет меньше по размеру, чем совокупность исходных картинок.

Спрайты. Что это и для чего

Пример

  1. Скачайте себе архив и разархивируйте. Сохраните его на диск D:
  2. Перейдите в директорию ex1
  3. Выполните команду npm init
  4. Выполните команду npm install -D spritesmith-cli
  5. Выполните команду spritesmith -rc .spritesmith.js
Путь к spritesmith нужно указывать относительно папки node_modules: ./node_modules/spritesmith-cli/bin/spritesmith

Полезные команды sass

      // Скомпилировать styles.scss -> styles.css
      sass styles.scss styles.css
      // Скомпилировать styles.scss -> styles.css
      // --style=expanded - расширенное форматирование
      sass --style=expanded styles.scss styles.css
      // Скомпилировать styles.scss -> styles.css
      // --sourcemap=none - отключить sourcemaps
      sass --style=expanded --sourcemap=none styles.scss styles.css

Полезные команды sass

      // Скомпилировать styles.scss -> styles.css
      // --style=compressed - минифицировать стили
      sass --style=compressed --sourcemap=none
      styles.scss styles.css

Полезные команды postcss-cli

      postcss -c postcss.config.js css/*.css -d css -r

Содержимое postcss.config.js 

      module.exports = {
        plugins: [
          require("autoprefixer")()
        ]
      };
Больше информации по postcss-cli плагину тут.

npm скрипты

      // <script-name> - имя скрипта
      // определенного в секции "scripts"
      npm run <script-name>
Больше информации по npm run <script-name> тут.

npm скрипты

Пример

      "scripts": {
        "sass-dev": "sass --style=expanded --sourcemap=file
              sass/styles.scss css/styles.css"
      }
    
      // вызов скрипта
      npm run sass-dev
