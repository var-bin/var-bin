Цвета в CSS. RGB vs. RGBA

      {
      /* integer range 0 - 255 */
        background-color: rgb(255, 0, 0);
      }
    
      {
      /* the same, with explicit opacity of 0.1
      range 0 - 1*/
        background-color: rgba(255, 0, 0, 0.1);
      }

Селекторы. Специфичность селекторов

  • a = 1 если стили заданы через атрибутstyle="". Иначе = 0.
  • b = считает количество ID в селекторе
  • c = считает количество классов, атрибутов и псевдо-классов в селекторе
  • d = считает количество селекторов по типу (тегов) и псевдо-элементов в селекторе
