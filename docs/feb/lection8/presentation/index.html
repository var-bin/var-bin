Для чего был придуман еще один режим разметки?

Терминология

  • main-axis - главная ось, вдоль которой располагаются flex-элементы.
  • main-start | main-end - flex-элементы размещаются в контейнере от позиции main-start до позиции main-end.

Терминология

  • main size - ширина или высота flex-элемента в зависимости от выбранной основной величины. Основная величина может быть либо шириной, либо высотой элемента.

Терминология

  • cross axis - поперечная ось, перпендикулярная к главной. Её направление зависит от направления главной оси.
  • cross-start | cross-end - flex-строки заполняются элементами и размещаются в контейнере от позиции cross-start и до позиции cross-end.

Терминология

  • cross size - ширина или высота flex-элемента в зависимости от выбранной размерности равняется этой величине. Это свойство совпадает с width или height элемента в зависимости от выбранной размерности.

Как определить flex контейнер

      display: flex | inline-flex;
  • display: flex; - создает flex-контейнер блочного типа.
  • display: inline-flex; - создает flex-контейнер inline типа.
Пример

Как определить flex контейнер. Важно знать

  • column-* CSS-столбцы
  • float, clear не создают плавающих flex-элементов и не "очищают" float
  • Псевдоэлементы::first-line и ::first-letterне применяются к flex-контейнеру

Определяем поведение flex контейнера

  • flex-wrap: nowrap* | wrap | wrap-reverse; - определяет, будет ли flex-контейнер однострочный или многострочный, а также направление по поперечнай оси.
  • flex-direction: row* | row-reverse | column | column-reverse; - определяет, каким образом будут расположены flex-элементы внутри flex-контейнера относительно главной оси.
  • flex-flow: <'flex-direction'> || <'flex-wrap'>
    * - значение по умолчанию

Определяем как будет распределятся пространство относительно главной оси

  • justify-content: flex-start* | flex-end | center | space-between | space-around; - распределяет flex-элементы вдоль главной оси на текущей линии flex-контейнера.
    * - значение по умолчанию
Пример

Определяем как будет распределятся пространство относительно поперечной оси

  • align-items: flex-start | flex-end | center | baseline | stretch*; - выравнивает все flex-элементы внутри flex-контейнера, включая анонимные flex-элементы.
    * - значение по умолчанию
Пример

Дополнительные свойства для выравнивания контента

  • align-self: auto* | flex-start | flex-end | center | baseline | stretch; - позволяет переопределить поведение align-items для конкретного flex-элемента.
    * - значение по умолчанию
Пример

Дополнительные свойства для выравнивания контента

  • align-content: flex-start | flex-end | center | space-between | space-around | stretch*; - позволяет задавать выравнивание для flex-линий внутри flex-контейнера, если есть свободное пространство. Выравнивание происходит относительно поперечной оси.
    * - значение по умолчанию

Управление размером flex элементов

  • flex-grow: 0* | number; - определяет flex-grow фактор, который в свою очередь определяет как flex-элемент(ы) будет увеличиваться относительно других flex-элементов. Не принимает отрицательные значения.
    * - значение по умолчанию
Пример

Управление размером flex элементов

  • flex-shrink: 1* | number; - определяет flex-shrink фактор, который в свою очередь определяет как flex-элемент(ы) будет уменьшаться относительно других flex-элементов. Не принимает отрицательные значения.
    * - значение по умолчанию
Пример

Управление размером flex элементов

  • flex-basis: auto* | content | width; - определяет начальный размер flex-элемента перед тем, как свободное пространство будет распределено с учетом flex-фактора.
    * - значение по умолчанию
Пример

Определяем порядок следования flex элементов

  • order: 0* | integer; - определяет в каком порядке будут появляться flex-элементы внутри flex-контейнера. Может принимать отрицательные значения
    * - значение по умолчанию
